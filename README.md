# Sistema de Biblioteca

Sistema de gerenciamento de biblioteca desenvolvido em Java utilizando Programação Orientada a Objetos (POO), JDBC e MySQL.

## Funcionalidades

- Cadastro de usuários
- Login de usuários
- Cadastro de livros
- Listagem de livros
- Empréstimo de livros
- Devolução de livros
- Controle de administradores e alunos

## Tecnologias Utilizadas

- Java
- Maven
- MySQL
- JDBC

## Estrutura do Projeto

- Entities
- Model
- Controllers
- Views

## Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/VictorGabriel13/biblioteca-java.git
   ```

2. Configure o banco de dados MySQL.

3. Execute os scripts SQL.

4. Execute a aplicação.
